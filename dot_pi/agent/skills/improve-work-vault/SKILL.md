## Linking
- When we're done with a folder, look through the rest of the vault topics
    - If there are specific notes with conceptual similarity, link them with obsidian style links
        - The convention is `[[note1]] [[note2]]`
- Be fairly stingy with this:
    - Linking something like "general-machine-learning.md" to "general-statistics.md" is so vague as to be more noise than signal
